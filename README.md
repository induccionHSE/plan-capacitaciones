# Control del Plan Anual de Capacitaciones
 
Sistema integral para la planeación, ejecución, evaluación y seguimiento del plan anual de capacitaciones del SG-SST. Incluye control de asistencia, evaluaciones de aprendizaje, encuestas de satisfacción, alertas de vencimiento y estadísticas de participación en Looker Studio.
 
> **v1.0 — Producción**
 
---
 
## Funcionalidades
 
| Módulo | Descripción |
|---|---|
| **Plan anual** | Programación de capacitaciones por mes, tema, responsable y población objetivo |
| **Registro de asistencia** | Captura por Google Forms con repositorio consolidado en Sheets |
| **Evaluaciones** | Formulario de evaluación de conocimientos por capacitación |
| **Satisfacción** | Encuesta de satisfacción post-capacitación con análisis automático |
| **Estadísticas de participación** | Base de datos que extrae y consolida respuestas para Looker Studio |
| **Alertas de vencimiento** | Notificación semanal automática de capacitaciones vencidas o por vencer |
| **Dashboard ejecutivo** | Tablero en Looker Studio con indicadores de cumplimiento del plan |
 
---
 
## Stack
 
- **Frontend:** Google Forms
- **Backend:** Google Apps Script
- **Triggers:** Semanal automático para alertas
- **Base de datos:** Google Sheets
- **Visualización:** Looker Studio
- **Notificaciones:** Gmail
